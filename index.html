Pokédex

The purpose of this website is to demonstrate my proficiency in working with an API, HTML, CSS, and Vanilla JavaScript. The API chosen is the RESTful Pokémon API found at https://pokeapi.co/.

I know this is nothing original since a lot of bootcamps promote this type of project, including mine, but I guess it's better to look at any content as opposed to placeholders. The intent is for a recruiter to learn a little about pokemon while at the same time realizing I am worth an interview. There exist two more versions of this Pokédex in the wild built by me, one created with vanilla JavaScript and Bootstrap and another created with jQuery and Bootstrap. They live here.
